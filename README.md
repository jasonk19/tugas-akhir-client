# Essay Analyzer System

## How to run

1. Make sure you have Node JS installed
2. Clone this repository
3. Navigate to this repo directory
4. Install dependencies
   ```
   npm install
   ```
5. Configure env file to supply api url
6. Run the application
   ```
   npm run dev
   ```
